# Day 1: Welcome & Tools

**Monday, July 13**

Today you'll set up your computing environment and make your very first physics histogram. By the end of the afternoon you'll have written real Python code that loads data and produces a plot.

---

## What you'll do

- Set up access to Jupyter notebooks via Google Colab or JupyterHub
- Fork the MESH repository to your own GitHub account
- Learn the basic git workflow (status → add → commit → push)
- Make a histogram of particle energies with Python

## What you'll learn

- How to run a Jupyter notebook
- What a histogram is and why physicists love them
- How to use `numpy` and `matplotlib` for basic data analysis
- How to save your work with git
