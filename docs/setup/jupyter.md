# Running Jupyter Notebooks

EPIC will use **Jupyter notebooks** for the computing sessions. The final notebook repository and launch instructions will be added once the notebooks are ready.

During the program, instructors will walk everyone through:

1. Opening Jupyter
2. Finding the notebook for the session
3. Running notebook cells
4. Saving work
5. Pushing work to GitHub

---

## Notebook basics

Once a notebook is open, these are the main actions:

| Action | How |
|--------|-----|
| Run a cell | Click it, then press **Shift + Enter** |
| Run all cells from the top | Use the notebook's **Run all** option |
| Add a new cell | Use the **+** or **Insert** controls |
| Stop a running cell | Use the stop/interruption control in Jupyter |

!!! tip "Read before you run"
    Each cell has text explaining what it does. Read it before pressing Shift + Enter; it will help you understand the analysis.
