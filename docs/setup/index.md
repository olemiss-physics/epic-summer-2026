# Setup Overview

All the computing in EPIC happens inside **Jupyter notebooks** — interactive documents that mix text, code, and plots. You run them in your browser, so there's nothing complicated to install.

This section will eventually walk you through two things:

1. Running notebooks — how to open and use the notebooks for this program
2. **[Using GitHub](github.md)** — how to save your work so you can come back to it

!!! tip "We'll do this together on Day 1"
    You don't need to complete this before the program. We'll walk through setup as a group on Monday morning. This page is here so you can get a head start if you want — or come back later if you need a reminder.
