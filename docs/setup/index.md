# Setup Overview

All the computing in EPIC happens inside **Jupyter notebooks** — interactive documents that mix text, code, and plots. You run them in your browser, so there's nothing complicated to install.

This section walks you through two things:

1. **[Running notebooks](jupyter.md)** — how to open and use the notebooks for this program
2. **[Using GitHub](github.md)** — how to save your work so you can come back to it

!!! tip "We'll do this together on Day 1"
    You don't need to complete this before the program. We'll walk through setup as a group on Monday morning. This page is here so you can get a head start if you want — or come back later if you need a reminder.

---

## Quick start

Here's the shortest possible path to getting a notebook open:

1. Go to [Google Colab](https://colab.research.google.com) (you'll need a Google account)
2. Click **File → Open notebook → GitHub**
3. Enter the MESH repository URL: `https://github.com/YOUR-GITHUB-ORG/MESH`
4. Choose the notebook for today
5. Click the play button on the first cell — you're coding!

See the [Running Notebooks](jupyter.md) page for step-by-step screenshots and more options.

## What you'll need

| Requirement | Details |
|-------------|---------|
| A browser | Chrome works best. Firefox and Edge are fine too. |
| A Google account | For Google Colab. If you don't have one, you can [create a free account](https://accounts.google.com/signup). |
| A GitHub account | For saving your work. [Sign up free](https://github.com/signup) if you don't have one. |

That's it. No software to install.
